#### Installation

Copy *mpv* folder to the following destination:
```
C:\Users\User-Name\AppData\Roaming
```
or 

Open the *Run dialog box* by pressing *Win + R*, then type *%appdata%*, and press *Enter*.
```
%APPDATA%
```
